App FAQ

Do you have problems using our app? We created this FAQ to help you solve them.

What is List?

We usually say list when we want to gather something into one place. We have 3 types of list in the app for now which is: Reminder, Countdown, Note.

What is Item?

Item is something that we put into the list. For example when you add a reminder list called “todo”, by adding an item you will actually add a new task inside your list which could have due date.

What is differences between lists?

So, we have 3 types of list. 1-Reminder, 2-Countdown, 3-Note. Countdowns have counting down label that shows you how long does your countdown take to reach the due date. Reminders are similar to countdowns but without the counting down labels. Notes are very simple and just have the title and description.

How do I add an Item?

There are two ways to add items. You can either hold the list you want to add an item into it and select “quick add item” or just tap the add item or quick add item button on the right side of the app.

How can I finish/unfinished an item?

There are two ways to finish an item. You can either swipe right and select finished/unfinished or hold the item and select finished/unfinished. You then see all your finished tasks at the end of the list.

What is differences between Finished and Unfinished?

Finished mean that your task or item is finished, however unfinished is the opposite of finished.

How can I set a reminder?

Whenever you want to add an item into a list you can select the “Add more detail” to go to the page that shows reminder and set the reminder there.

Why the add item color changes?

When you select all items list or importants list the color of add item changes to green. You wont quick add an Item because we don’t know which list do you want to add item in.

How can I change the list of an Item? Change date? Edit the Item?

Hold on the item and tap edit. You can edit everything about the Item in there.

How can I delete a task?

Hold on the item and tap delete.

How can I create a new list?

Tap the green button “+ add list" in the main view at the left bottom of the screen.

How can I edit or delete the list?

Simply tap and hold on the list and select delete or edit.

How can I change the application language?

You can select the top right menu button and go to setting, then select your language.

How can I get rid of advertisements?
